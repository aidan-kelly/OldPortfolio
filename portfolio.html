Conway's Game of Life

A basic implementation of Conway's Game of Life written in Python. It randomly generates a board of a size that you specify and then follows the rules outlined on the Wikipedia Page. In the future I would like to recreate the Game of Life in Javascript with the ability to create custom inputs instead of relying on a random input.

Photoframe Application

This application allows you to remotely upload photos to a Google Drive, and then have the photos display on a monitor connected to a Raspberry Pi running this application. It was meant to be a "smart" photoframe gift for my family.
